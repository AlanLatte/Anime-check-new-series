# How use:
> Step 1:
>> Clone repo
```sh
	$ git clone https://github.com/AlanLatte/Anime-check-new-series.git
```
> Step 2:
>> Run app/main.py
```sh
	$ python3 app/main.py
```
> Step 3:
>> Add to urls.txt any urls (sites: anilibria or anidub)
> Step 4:
>> Run again app/main.py
```sh
	$ python3 app/main.py
```
